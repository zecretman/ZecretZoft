Requirement
===========

- NW.js-0.14.4 : http://dl.nwjs.io/v0.14.4/nwjs-v0.14.4-linux-x64.tar.gz
- Node.js-5.11.1 : https://nodejs.org/download/release/v5.11.1/

npm install nw-gyp
npm install sqlite3 --build-from-source --runtime=node-webkit --target=NW_VERSION
npm install async --build-from-source --runtime=node-webkit --target=NW_VERSION
